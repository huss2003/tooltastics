## Introduction

Reading a ruler seems simple — until you're trying to figure out what the tiny marks between the big numbers actually mean. Whether you're using a physical ruler or RulerKit's online ruler, understanding the markings on both the imperial (inches) and metric (centimeters) sides is a foundational skill for anyone who builds, designs, or crafts.

This guide breaks down both measurement systems so you can read any ruler with confidence.

## The Two Sides of a Ruler

Most rulers display two scales:

- **Metric side** (centimeters and millimeters)
- **Imperial side** (inches and fractions of an inch)

RulerKit's online ruler includes both scales, and you can toggle between them with a single click.

## How to Read Centimeters (Metric)

The metric system is straightforward because it's decimal-based.

### Markings on the Metric Side

| Mark | Meaning | Length |
|---|---|---|
| Longest line with a number | Centimeter (cm) | 10 mm |
| Medium line | Half-centimeter | 5 mm |
| Shortest line | Millimeter (mm) | 1 mm |

### Step-by-Step: Reading cm

1. **Find the centimeter mark** closest to your object (but before its end). Each numbered mark is 1 cm.
2. **Count the small lines** after that centimeter mark. Each small line is 1 millimeter.
3. **Add them together**. For example: 5 cm + 3 small lines = 5.3 cm (or 53 mm).

> **Pro tip:** There are 10 mm in 1 cm. If you're measuring in mm, just count all the small lines from zero.

### Example

If an object starts at 0 and ends at the 5th small line after the 4 cm mark:

```
4 cm + 5 mm = 4.5 cm (or 45 mm)
```

## How to Read Inches (Imperial)

The imperial side is trickier because inches are divided into fractions.

### Common Fractional Divisions

Most rulers divide each inch into:

- **1/2 inch** — the longest line between inch marks
- **1/4 inch** — the next longest line
- **1/8 inch** — shorter still
- **1/16 inch** — the shortest line on most rulers

Some precision rulers also include **1/32 inch** markings.

### Step-by-Step: Reading Inches

1. **Identify the nearest whole inch** — look for the largest numbered mark before your object's end.
2. **Find the longest line** between inch marks — that's 1/2 inch.
3. **Count the smaller lines** to determine the fraction:
   - The 1/2 mark splits the inch in two
   - Each quarter splits further
   - Eighths and sixteenths follow the same pattern

### Quick Reference: Inch Fractions to Decimals

| Fraction | Decimal | Common Use |
|---|---|---|
| 1/16" | 0.0625 | Fine woodworking |
| 1/8" | 0.125 | General carpentry |
| 1/4" | 0.25 | Hardware spacing |
| 3/8" | 0.375 | Socket sizes |
| 1/2" | 0.5 | Standard spacing |
| 5/8" | 0.625 | Construction |
| 3/4" | 0.75 | Plumbing |
| 7/8" | 0.875 | Framing |
| 1" | 1.0 | Base unit |

### Reading to 1/16 of an Inch

On a standard ruler, the 1/16" marks are the smallest. Count them like this:

```
0    1/4    1/2    3/4    1
|  |  |  |  |  |  |  |  |
1/16  3/16  5/16  7/16
  2/16  4/16  6/16  8/16
```

Remember that fractions should be reduced: 2/16 = 1/8, 4/16 = 1/4, 8/16 = 1/2.

## Common Mistakes to Avoid

- **Not starting at zero** — ensure the object is aligned with the 0 mark, not the edge of the ruler
- **Parallax error** — view the ruler straight on, not from an angle
- **Miscounting fractions** — always reduce fractions to their simplest form
- **Confusing cm and inch sides** — double-check which scale you're reading
